# DressMe MVP v1

## Objective

Validate that a fashion business can run entirely on DressMe.

The admin should manage products, inventory, branding and orders without touching the database.

Customers should discover products, place orders and communicate through WhatsApp.

---

## Customer Features

- Landing page
- Product browsing
- Product search
- Category browsing
- Brand browsing
- Price filter
- Product details
- Multiple product images
- Wishlist
- Shopping cart
- Checkout
- Address management
- Order history

- WhatsApp order confirmation
- AI Stylist Lite
